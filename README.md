# 黄慧凝的个人简历网站

这是我的个人简历网站，使用 HTML、CSS 和 JavaScript 构建。

## 特点

- 响应式设计，适配各种设备
- 现代简约的界面
- 流畅的动画效果
- 清晰的内容结构

## 技术栈

- HTML5
- CSS3
- JavaScript
- CSS Animations
- Intersection Observer API
